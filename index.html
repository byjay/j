아빠의 마음

"우리 가족과 함께하는 첫 일본 여행! 설레는 마음으로 직접 만들었어요. ❤️"

🏠 홈 탭

  • • 4가지 메인 메뉴 (글자, 회화, 단어, 여행) 바로가기
  • • 앱 설치 버튼 (PWA 지원)
  • • 도움말 버튼

📝 글자 탭

  • • 히라가나/가타카나 전체 글자 그리드
  • • 글자 클릭 → 쓰기 연습 (획순 애니메이션)
  • • 퀴즈: 10문제 랜덤 출제, 90점 이상 2회 시 여행탭 잠금 해제
  • • 통계: 취약 글자, 정답률 확인

💬 회화 탭

  • • 상황별 회화: 공항, 식당, 쇼핑 등 상황별 예문
  • • 실전 회화: 카드 뒤집기로 일본어↔한국어 학습
  • • 발음 듣기 버튼 (TTS)

📚 단어 탭

  • • 카테고리별 단어장 (인사, 숫자, 음식 등)
  • • 단어 카드 뒤집기 학습
  • • 발음 듣기 및 예문 확인

✈️ 여행 탭

  • • 후쿠오카 4일 일정: 맛집, 관광지, 쇼핑 정보
  • • 장소별 상세정보 (영업시간, 메뉴, 팁, 지도)
  • • 구글맵 사진 링크: 실제 장소 사진 검색
  • • 일정 커스터마이징 (장소 추가/삭제)

📊 현황 탭

  • • 연속 학습(Streak): 매일 학습 시 +1일
  • • 레벨 & XP: 학습 활동마다 경험치 획득
  • • 주간 차트: 7일간 학습량 그래프
  • • 퀴즈 성적: 최근 10회 성적 그래프
  • • 용돈 모으기: 1000XP+공부시간 180분+퀴즈 100점 3회 = 2만원!

🔓 여행지 잠금 해제 조건

  • ✅ 후쿠오카: 기본 제공
  • 🔒 오사카/교토: 글자 퀴즈 90점 2회 달성
  • 🔒 도쿄: 총 경험치 500XP 달성

🌟 특별 기능

  • 💰 용돈 청구: 시은/하롱 전용, 조건 달성 시 아빠에게 청구!
  • 💃 춤추는 캐릭터: 퀴즈 100점 시 캐릭터 춤 애니메이션
  • 📱 앱 설치: 홈 화면에 추가하면 앱처럼 사용
  • 🔔 iOS 안내: Safari '공유' → '홈 화면에 추가'
